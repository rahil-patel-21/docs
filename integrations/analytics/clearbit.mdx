---
title: "Clearbit"
---

Add the following to your `mint.json` file to send analytics to Clearbit.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Example
"analytics": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
